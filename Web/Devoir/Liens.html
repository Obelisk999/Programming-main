Liens


Html (Hyper Text Markup Language) est un langage hypertexte (et hypergraphique)
qui vous permet en cliquant sur un mot, généralement souligné (ou une image) de vous
Transporter;
  • vers un autre endroit du document.
  • vers un autre fichier Html situé sur votre ordinateur.
  • vers un autre ordinateur situé sur le Web.

Ce système d'hypertexte vous est familier car il est également utilisé par les fichiers
d'aide de Windows. Ce sont ces liens qui vous permettent de surfer de page en page
et qui constituent l'essence des documents Html.

  • La syntaxe de ces liens entre plusieurs pages, est simple mais entraînera de nombreux
    commentaires :

<A HREF="URL ou adresse">...</A>
Exemple:
<A HREF="exempleliste.html">suivant</A>
